Technetium

What is Technetium?

Technetium is a data aggregation web application built using Django on top of the Bitbucket API. Through this scalable API, Technetium is able to deliver simplistic yet highly useable tools perfect for use in any development environment.

Why Technetium?

Repository Manager



View all read-access enabled repositories on your account through Technetium's repository manager. Using its simplistic yet powerful accordion-style drop down lists, subscribe to repositories that you'd like to keep up with.

Centralized Issue Tracker



View issues acrossa multitude of repositories using Technetium's central issue tracker. From bug tracking to resolved issues, our centalized issue tracker is perfect for all members of the development team to keep up with the project.

Statistics/Graphs

View commit/issue activity in both tabular and graphical views using Technetium's report module. Perfect for product owners and scrum masters to track the progress of any project.
