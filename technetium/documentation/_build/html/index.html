bitstats.list_data(data, key_value='changesets')¶

Returns a list of data in order of developers. Useful for D3 graphs. You can use this to list commit data as well as issues.

Paramters:
data: Dictionary

  • A list containing the number of commits for each user of a given repository.

bitstats.tally_changesets(data)¶

Counts the number of commits for each developer in a repository.

Parameters:
data: Dictionary
  • A dictionary containing the commits to be tallied.
Returns:
tally: Dictionary
  • A dictionary containing the sums of all commits in the repository.
Example:
{ ‘David Leonard’ : {‘changesets’ : 9}, ...}

Note: The client and client_secret come from the settings.

Returns:
OAuth1 (object)

Gets a dictionary of the following data from social auth:

  3. ‘first_name’ : String
